# xargs

xargs 命令是一个其他命令参数传递一个过滤器,擅长将stdin转化成命令参数,xargs 能够处理管道或者 stdin 并将其转化成特定命令的命令参数.

xargs 的默认命令是echo ,空格是默认定界符.
这也就意味着通过管道传递给xargs包含换行和空白,不过通过xargs的处理,换行和空白将被空格取代

cat abc | xargs echo <=====> cat abc | xargs
